适用于团队的 Creative Cloud 是一个创意平台，将用户、应用、硬件、学习内容与社区联系起来。它向中小型企业客户提供创作最佳作品所需的创意工具，同时使软件管理变得快速、轻松和灵活。

高级代码完成和验证
利用可将简单缩写转换为复杂 HTML 和 CSS 代码的 Emmet 支持以及 Linting 实时错误检查，更高效地编写代码。您还可以在“实时视图”下编辑 PHP 文件中嵌入的 HTML 代码。

Photoshop 画板支持

Dreamweaver CC 2018版本新增功能概述

支持 Bootstrap 4.0
Dreamweaver 的最新版本提供对 Bootstrap 4.0.0 版本的支持。通过 Bootstrap 4.0.0，您可以利用备受欢迎 CSS 框架的所有新增功能，同时在 Dreamweaver 中构建响应式站点。
借助 Dreamweaver 中的 Bootstrap 4.0.0，您能够使用“卡片”、“徽章”和“表单”等组件。此外，您可以访问 “代码片段” 面板中的 4.0.0 文件夹，它位于 “Bootstrap 代码片段”下面。

Dreamweaver 中的 HiDPI 显示器支持 (Windows)
现在，Dreamweaver CC 为 Windows 提供 HiDPI 显示器支持，并且可以根据显示器识别 DPI。也就是说，应用程序会根据显示器的缩放设置进行缩放。|
之前，在 4k 显示器中，Dreamweaver 中的 UI 元素无法正确地缩放。菜单、字体、图标等项目都存在像素化问题，并且看起来很小很模糊。利用对 HiDPI 的支持，Dreamweaver 改善了用户体验。UI 元素看起来更加清晰，且不会出现像素化现象。此外，您不会遇到 UI 元素溢出等问题。

Dreamweaver 中的多显示器支持 (Windows)
现在，Dreamweaver CC 为 Windows 提供多显示器支持。应用程序会根据显示器的分辨率自动缩放。
多显示器支持提供了多项好处。例如，您可以让某个文档不再以选项卡的形式呈现，而是在另一个显示器中查看该文档。通过这样做，您可以轻松打开和对比某一文档的不同版本，或者打开和对比两个不同的文档。
您还可以在多个显示器中查看和编辑不同类型的 Dreamweaver 文档。例如，您可以在一台显示器中编辑 HTML 文档，同时在另一台显示器中查看或编辑 CSS 文档。

Git 支持的增强功能
您现在可以在 Dreamweaver 中检查远程 Git 存储库的连接性。单击“站点设置”对话框或“添加新远程”对话框中的“测试”。Dreamweaver 会测试您的远程存储库的 URL 是否有效。

保存凭据
Dreamweaver 现在允许您在克隆存储库时保存凭据。当您在“站点设置”对话框中输入凭据时，您可以保存凭据。您不需要在每次执行推送和提取等远程 Git 操作时都重新输入凭据。
之前，“保存凭据”选项位于“添加远程”和“管理远程”对话框中。每当需要更新您的凭据时，您都必须转到这些对话框并重新输入凭据。现在，您可以在克隆存储库时直接保存凭据。

在 Git 面板中搜索文件
利用 Dreamweaver，您可以在“Git”面板中搜索文件。作为用户，当您的存储库中包含大量文件时，您可以使用文件名搜索文件。

当您在“搜索”框中输入文件名后，系统会立即显示所有包含搜索短语的文件。您还可以在搜索结果中查看这些文件的状态，从而帮助您跟踪已暂存、已修改和未跟踪的文件。

更多新增功能请查阅官网资料：https://helpx.adobe.com/cn/dreamweaver/using/whats-new.html
